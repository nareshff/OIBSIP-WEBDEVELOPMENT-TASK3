TASK 3=> Temperature Converter

Objective

The main objective of this project is to design and develop a Temperature Converter web application using HTML5, CSS3, and JavaScript. The application allows users to convert temperature values between Celsius (°C), Fahrenheit (°F), and Kelvin (K) accurately and instantly. It provides a simple, user-friendly interface that enables users to enter a temperature value, select the input unit, and view the converted values in the remaining temperature units.

This project aims to improve the understanding of front-end web development concepts such as HTML structure, CSS styling, JavaScript functions, user input validation, DOM manipulation, event handling, and mathematical calculations. The application is designed with a clean layout, responsive design, and attractive user interface to provide a smooth user experience.

Steps Performed

The following steps were performed during the development of the Temperature Converter project:

Planned the overall layout of the web application.
Created the basic HTML structure using HTML5.
Added a heading for the application.
Created an input field to accept the temperature value from the user.
Added a dropdown menu allowing users to select Celsius, Fahrenheit, or Kelvin.
Created a Convert button to perform the conversion.
Added a result section to display the converted temperature values.
Linked an external CSS file to style the webpage.
Designed the interface using CSS with a gradient background, centered container, rounded corners, and shadows.
Styled the input field, dropdown menu, button, and result section for better readability.
Wrote JavaScript code to perform temperature conversions.
Implemented input validation to prevent empty or invalid values.
Used JavaScript DOM methods to read user input and display results dynamically.
Applied mathematical formulas for converting temperatures between Celsius, Fahrenheit, and Kelvin.
Displayed converted values with two decimal places for improved accuracy.
Tested the application with different temperature values to verify correct output.
Ensured the application provides accurate and user-friendly results.
